What is Programming?

Before we start, we need to first understand what programming is.

Programming is the process of creating software or applications that can perform specific tasks. It involves writing code, which is a set of instructions that a computer can understand and execute.

Think of it like giving a set of instructions to a robot to perform a certain task. The robot can only do what it has been programmed to do, and programming is the process of creating instructions for the computer to follow.

Programming can be broken down into 5 core building blocks. Knowing these is not enough, but they are the foundation everything else is built upon.

In the sections below are summaries of all these 5 building blocks.

variables

In Python, a variable is a named reference to a value that can be stored in the computer's memory. When we assign a value to a variable, we are telling the computer to reserve a portion of its memory to store that value, and to associate that value with the variable name.

            
              x = 5
              y = "Hello, world!"
              z = [1, 2, 3, 4]
              student = {"name": "John", "age": 25}

For example, in the code block above, the line x = 5 creates a variable named x and assigns it the value 5. The computer reserves a portion of its memory to store the value 5, and associates that value with the variable name x.

When we refer to a variable later in our code, Python looks up the value associated with that variable name in memory, and uses that value in our code. For example, if we later write y = x + 2, Python will look up the value associated with the variable name x (which is 5), add 2 to it, and then store the result in a new variable named y.

It's important to note that variables are not the same as the values they store. When we assign a value to a variable, we are telling the computer to reserve memory for that value and associate it with the variable name. If we later assign a new value to that same variable, the old value is not deleted or overwritten - it is simply "forgotten", and the variable is associated with the new value instead.

Overall, understanding how variables work is important for understanding how Python (and other programming languages) manage and manipulate data in computer memory.

In the examples above, x is a variable that stores the integer value 5, y is a variable that stores the string value "Hello, world!", z is a variable that stores a list of integers, and student is a variable that stores a dictionary with two key-value pairs.

You can also perform operations with variables, for example:

            
              a = 10
              b = 20
              c = a + b
              print(c)
data types

In Python, several built-in data types can be used to represent different kinds of data. Here are some of the most common data types:

- Numbers: Python supports different types of numbers, including integers (int), floating point numbers (float), and complex numbers (complex). Examples:

            
              x = 5          # integer
              y = 3.14       # float
              z = 2 + 3j     # complex

- Strings: A string is a sequence of characters. In Python, strings can be enclosed in single quotes ('...') or double quotes ("..."). Examples:

            
              s1 = 'Hello, world!'
              s2 = "Python is awesome"

- Lists: A list is a collection of values that are ordered and mutable (i.e., can be changed). In Python, lists are enclosed in square brackets ([...]). Examples:

            
              my_list = [1, 2, 3, 4, 5]
              my_list2 = ["apple", "banana", "cherry"]

- Tuples: A tuple is a collection of values that are ordered and immutable (i.e., cannot be changed). In Python, tuples are enclosed in parentheses ((...)). Examples:

            
              my_tuple = (1, 2, 3, 4, 5)
              my_tuple2 = ("apple", "banana", "cherry")

- Dictionaries: A dictionary is a collection of key-value pairs that are unordered and mutable. In Python, dictionaries are enclosed in curly braces ({...}). Examples:

            
              dict1 = {"name": "John", "age": 25, "city": "Douala"}
              dict2 = {"apple": 1, "banana": 2, "cherry": 3}

- Boolean: A Boolean value is either True or False. In Python, Boolean values are used for logical operations and conditions. Examples:

            
              x = True
              y = False

These are just a few examples of the most common data types in Python. Understanding data types is important for programming in Python, as it allows you to manipulate data in different ways depending on the type of data you are working with.

functions and methods

In Python, a function is a block of code that performs a specific task. Functions are used to organize code into reusable blocks, which can be called multiple times throughout a program. In Python, functions are defined using the def keyword, followed by the function name and any arguments the function takes.

Here is an example of a simple function that takes two arguments and returns their sum:

            
              def add_numbers(x, y):
                result = x + y
                return result

In this function, x and y are the arguments that are passed to the function when it is called. The function then adds these two numbers together and stores the result in a variable named result. Finally, the function returns the value of result.

To call this function, we simply use the function name followed by the arguments in parentheses:

            
              print(add_numbers(3, 5))

This code will output 8, which is the result of calling the add_numbers function with arguments 3 and 5.

In Python, a method is a function that is associated with an object. Methods are similar to functions in that they perform a specific task, but they are called on an object instead of being called independently. Methods are defined in the same way as functions, but they are called using the dot notation.

Here is an example of a method that is associated with a list object:

            
              my_list = [1, 2, 3, 4, 5]
              my_list.reverse()
              print(my_list)

In this code, we create a list object named my_list that contains the values [1, 2, 3, 4, 5]. We then call the reverse method on the my_list object, which reverses the order of the list in place (i.e., without creating a new list). Finally, we print the contents of my_list, which now contains the values [5, 4, 3, 2, 1].

control flow

Control flow refers to the order in which code is executed in a program. In Python, control flow is controlled using conditional statements (if, elif, and else) and loops (for and while).

Conditional Statements

Conditional statements are used to execute different code blocks based on whether certain conditions are true or false. The basic syntax for an if statement in Python is:

            
              if condition:
                # code to execute if condition is True

The condition can be any expression that evaluates to a Boolean value (True or False). If the condition is True, the code block following the if statement will be executed. If the condition is False, the code block will be skipped.

Here is an example:

            
              x = 5

              if x > 0:
                print("x is positive")

In this example, the condition is x > 0. Since x is equal to 5, which is greater than 0, the code block following the if statement will be executed, and the output will be x is positive.

You can also use elif and else statements to execute different code blocks based on multiple conditions. The basic syntax for an if/elif/else statement in Python is:

            
              if condition1:
                # execute if condition1 is True
              elif condition2:
                # execute if condition1 is False and condition2 is True
              else:
                # execute if both condition1 and condition2 are False

Here is an example:

            
              x = 5

              if x < 0:
                print("x is negative")
              elif x == 0:
                print("x is zero")
              else:
                print("x is positive")

In this example, the condition1 is x < 0, the condition2 is x == 0, and the else statement covers any other cases. Since x is equal to 5, which is greater than 0, the condition1 is False. The condition2 is also False, so the else statement will be executed, and the output will be x is positive.

loops

In Python, loops are used to execute a set of statements repeatedly until a specific condition is met. There are two types of loops in Python: for loop and while loop.

For Loop

A for loop is used to iterate over a sequence (a string, a list, a tuple, a set, or a dictionary). The syntax of a for loop is as follows:

            
              for {variable} in {sequence}:
                {statement(s)}

Here, variable is the iteration variable that takes on the value of the next element in sequence each time the loop runs. The {statement(s)} block is executed once for each item in sequence.

For example, the following code prints each item in a list:

            
              fruits = ["apple", "banana", "cherry"]
              for x in fruits:
                print(x)

Using range() with a For Loop

The range() function in Python is often used in combination with a for loop to perform an action a specific number of times. The range() function generates a sequence of numbers, which can be used as the sequence for the for loop to iterate over.

Here is an example:

            
              for i in range(5):
                print(i)

In this example, the for loop will print the numbers 0 through 4, because range(5) generates the sequence of numbers [0, 1, 2, 3, 4].

You can also specify a step value to skip numbers in the sequence. The syntax for this is range(start, stop, step). Here is an example:

            
              for i in range(0, 10, 2):
                print(i)

In this example, the for loop will print the numbers 0, 2, 4, 6, and 8, because range(0, 10, 2) generates the sequence of numbers [0, 2, 4, 6, 8].

While Loop

A while loop is used to execute a block of code repeatedly as long as a certain condition is true. The syntax of a while loop is as follows:

            
              while {condition}:
                {statement(s)}

Here, {condition} is tested before each iteration of the loop. If it is true, the {statement(s)} block is executed. This process repeats until the {condition} is false.

            
              i = 0
              while i < 5:
                print(i)
                i += 1
